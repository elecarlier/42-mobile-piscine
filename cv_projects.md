# Projects — Mobile Development (React Native / Expo)

## Mobile Piscine — 42 School

**Tech stack:** React Native, TypeScript, Expo, REST APIs, Git

---

### Module 00 — UI Components & Navigation
Built a series of mobile UI exercises including a fully functional calculator app.
- Implemented custom UI components with React Native (Views, Text, TouchableOpacity)
- Managed application state with React hooks (useState)
- Configured multi-screen navigation using Expo Router

---

### Module 01 — Navigation & Dynamic Interfaces
Built a weather application shell with tab-based navigation.
- Implemented swipeable tab navigation using React Navigation (Material Top Tabs)
- Designed responsive layouts with dynamic data rendering
- Integrated user input handling (search bar, geolocation button)

---

### Module 02 — REST API Integration & Geolocation
Built a fully functional weather application consuming multiple REST APIs.
- Consumed REST APIs: open-meteo (weather forecast), OpenStreetMap Nominatim (reverse geocoding)
- Implemented device GPS geolocation using expo-location with runtime permission handling
- Built dynamic city autocomplete with real-time API calls as the user types
- Displayed current weather, hourly forecast (Today) and 7-day forecast (Weekly)
- Validated all API responses using Zod schema validation
- Handled edge cases: API failures, denied permissions, no network, unknown cities
- Cross-platform development targeting Android (tested on emulator)
